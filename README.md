# Seffer Global

A modern Next.js website for Seffer Global, a worldwide garment export company.

## Project Overview

Seffer Global is a trusted exporter of garments ranging from everyday wear to luxury fashion. This website showcases our product categories, company values, and provides contact information for potential business partners.

## Features

- **Hero Section**: Company branding with contact call-to-action
- **About Us**: Company overview and mission
- **Product Categories**: Showcase of garment types (T-Shirts, Formal Shirts, Dresses, Jeans & Trousers, Sportswear, Kidswear)
- **Why Choose Us**: Key selling points and competitive advantages
- **Contact Section**: Direct contact via email and WhatsApp
- **Responsive Design**: Optimized for all device sizes

## Tech Stack

- **Framework**: Next.js 15.2.8 with TypeScript
- **Styling**: Tailwind CSS with custom design system
- **UI Components**: Radix UI + shadcn/ui
- **Deployment**: Firebase App Hosting
- **Icons**: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:9002](http://localhost:9002) in your browser.

## Available Scripts

- `npm run dev` - Start development server with Turbopack on port 9002
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run typecheck` - Run TypeScript type checking

## Design System

- **Primary Color**: Navy Blue (#2E3192)
- **Accent Color**: Gold (#D4AF37)
- **Background**: Light Gray (#F0F0F0)
- **Typography**: Inter font family

## Deployment

This project is configured for Firebase App Hosting. The `apphosting.yaml` file contains the deployment configuration.
